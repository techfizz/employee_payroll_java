Employee Payroll System
This project implements a basic Employee Payroll System in Java, utilizing various Java Collections to manage employee data and process payroll. The system demonstrates core Object-Oriented Programming (OOP) principles and provides a foundation for more complex payroll management.
Features
Add Employees: Add new employees to the system with their details (ID, name, salary/hourly rate).
Manage Employee Types: Support for different employee types (e.g., Salaried, Hourly) with distinct payroll calculation logic.
Calculate Payroll: Compute the salary or wages for each employee based on their type and relevant data.
View Employee Details: Display information for individual employees or a list of all employees.
Utilize Java Collections: Efficiently store and manage employee objects using ArrayList, HashMap, or other suitable collections.
Technologies Used
Java Development Kit (JDK) 8 or higher
Setup and Running
Clone the repository.
Code

   git clone <repository_url>
Navigate to the project directory.
Code

   cd employee-payroll-system
Compile the Java source files.
Code

   javac *.java
Run the main application.
Code

   java EmployeePayrollSystem
